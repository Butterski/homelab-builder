# Homelab Builder

MVP application for configuring hardware for homelab services.

## Stack
- React + Vite (Frontend)
- Go + Gin (Backend)
- PostgreSQL

## MVP Features
1. Service catalog with RAM/CPU/storage requirements
2. User service selection
3. Hardware recommendations generation
4. Shopping list with links
5. Mermaid diagrams
6. Google authentication
